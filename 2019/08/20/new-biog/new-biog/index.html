4.JAVA培训 第二天下午 java 的继承与派生 java的多态

  • 类似于QT 所有类都继承与某一个大类
    QT是继承于MainWindow
    JAVA是继承于object
  • 构造函数则和C++一样
    类似于虚函数
    自己不写构造函数的话 编译器会自动添加一个无参构造函数
  • JAVA 继承的关键字 extends
  • JAVA只允许单继承 不允许多继承 但允许多个接口
  • 继承时同名的方法
    子类优先(重写)
    重写规则 参数和返回值一样
  • 重写和重载的区别
    变量不能被重写(覆盖) 方法才能被重写
  • 多态
    条件 ： 继承 –> 重写 –> 父类引用指向子类对象
  • 当父类和子类有相同的变量时
    父类和子类的内存空间都会创建且父类和子类的属性都会保留，当父类和子类有相同名字的变量时，父类的变量隐藏
  • 一般不直接通过x.bb来取值，而是通过一个函数来取值
  • 鲁棒性
  • 对变量在类中赋值 this.xxx = xxx 不加this会有警告
  • java包 创建
